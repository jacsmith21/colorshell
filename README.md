# colorshell

A simple package to enable printing in color in IDLE. Works for python 3.

### Installation

```
pip install colorshell
```

## Example Usage

```
from colorshell import out
out.red('prints in red')
out.purple('print in purple')
```

## Supported Colors

1. black
2. red
3. green
4. purple
5. blue
6. orange
